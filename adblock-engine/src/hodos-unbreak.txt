! Title: Hodos Browser Compatibility Exceptions
! Description: Disables scriptlet injection, cosmetic CSS, and network blocking
!   on domains where it breaks authentication or core functionality
! Expires: 7 days
! Last modified: 2026-02-27
! License: MIT
! Homepage: https://github.com/ArcBit/Hodos-Browser
!
! NOTE: Cookie handling is separate from this file. Third-party cookies are
! managed by EphemeralCookieManager (Brave-style ephemeral storage) — cookies
! are allowed while a site has open tabs, then cleaned up 30s after the last
! tab closes. The old AUTH_COOKIE_DOMAINS hardcoded allowlist has been removed.
! This exception list handles adblock/cosmetic/scriptlet exceptions ONLY.

! =====================================================================
! Authentication Domains — adblock/cosmetic/scriptlet exemptions
!
! Each auth domain gets TWO layers of exceptions in this file:
!   1. #@#+js()      — disable scriptlet injection (fetch/XHR proxy breaks OAuth)
!   2. @@||...$generichide — disable cosmetic CSS hiding (EasyList hides login forms)
!
! Same-entity network exceptions (e.g. gstatic.com on google.com, twimg.com on
! x.com) are handled systemically by entity-aware blocking using the disconnect.me
! entity list — no per-domain @@|| network rules needed for same-org CDN resources.
!
! See auth-cookies-profiles-guide.md for the full rationale.
! =====================================================================

! --- Twitter / X ---
! OAuth + session cookies; fetch() proxy breaks token exchange.
! Cosmetic CSS filtering hides legitimate post images (EasyList selectors
! match organic tweet containers, not just ads).
! Network exceptions for same-entity CDNs (twimg.com, t.co) are handled
! systemically by entity-aware blocking (disconnect.me entity list).
x.com#@#+js()
twitter.com#@#+js()
api.twitter.com#@#+js()
abs.twimg.com#@#+js()
@@||x.com^$generichide
@@||twitter.com^$generichide

! --- Google Auth ---
! Google Sign-In, OAuth, FedCM, CAPTCHA.
! accounts.google.com is the primary auth page — needs scriptlet + cosmetic exceptions.
! Network exceptions for same-entity CDNs (gstatic.com, googleapis.com, etc.) are
! handled systemically by entity-aware blocking (disconnect.me entity list).
accounts.google.com#@#+js()
accounts.youtube.com#@#+js()
myaccount.google.com#@#+js()
@@||accounts.google.com^$generichide
@@||myaccount.google.com^$generichide

@@||login.live.com^$generichide
@@||login.microsoft.com^$generichide

! OAuth SDK Network Exceptions (global — needed on ANY site using these)

! =====================================================================

@@||accounts.google.com^$subdocument
