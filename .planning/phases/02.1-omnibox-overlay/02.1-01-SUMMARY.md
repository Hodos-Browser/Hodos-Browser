## Decisions Made

- **Cross-platform implementation**: Initially Windows-only, then added macOS support following wallet overlay pattern with NSWindow and OmniboxOverlayView
- **300px overlay height**: Just enough for dropdown, not full window height like other overlays
- **Unified pill container**: Single rounded Box wrapping input + dropdown (Chrome-like)
- **Auto-show dropdown on mount**: Chrome behavior when clicking address bar
- **Reuse Omnibox.tsx**: Avoided rebuilding by reusing existing component in overlay context
