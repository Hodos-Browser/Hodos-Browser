**Tech stack available:**
- React 19 with MUI 7 for UI
- Rust Actix-web backend on localhost:3301
- CEF V8 injection via window.hodosBrowser
- HTTP interception pattern established in HttpRequestInterceptor.cpp

**Established patterns:**
- useHodosBrowser hook pattern for wallet operations
- window.hodosBrowser.wallet.* API calls
- HTTP interception routes wallet API calls to Rust backend
- React components use async/await with error handling

**Constraining decisions:**
- No changes to Rust wallet backend (endpoints already exist)
- No changes to SQLite schema
- Follow existing frontend patterns (useWallet hook, MUI components)

**Rust endpoints available:**
- GET /wallet/balance - Returns current wallet balance
- POST /wallet/address/generate - Generates new receive address
- GET /wallet/address/current - Gets current receive address
- POST /transaction/send - Sends BSV transaction
</context>
