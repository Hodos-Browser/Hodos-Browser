## Test Methodology

### ✅ No Circular Validation

1. **Bitcoin Standard Tests**: Test against well-defined Bitcoin script standards (not our interpretation)
2. **Cross-Validation Tests**: Use TypeScript SDK (`@bsv/sdk`) as reference implementation
   - Generate scripts with TypeScript
   - Decode with Rust
   - Compare results
